Install MLX-Moxy-Wirks Page Assist

Load the extension from the standalone folder below (outside this repo) in any Chromium browser.

  1. Open chrome://extensions or edge://extensions.
  2. Enable Developer mode.
  3. Click Load unpacked.
  4. Select this folder:
/Users/bluewirks.max/Desktop/mlx-moxy-wirks-chromium-page-assist

Once installed, click the extension icon to open the side panel, then use it to:

  1. Send Current Page to capture title, URL, selected text, and page excerpt.
  2. Ask Moxy About This Page to run a grounded response against current page context.
  3. Optionally enable Agent mode to let Moxy take webpage actions (navigate, click, type) through the Playwright sidecar.
  4. Optionally set a System prompt to control response style and depth in the side panel.

If the side panel does not open on icon click, pin the extension and re-open it from the toolbar after pressing Reload on the extension card.
